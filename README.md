# itmo_naturalist_project

Учебный ML-проект. Аналитический сервис на открытых данных iNaturalist для Вьетнама: распознавание видов по фото, моделирование пространственно-временного распределения видов (SDM), миграционные коридоры, фенология, карточка вида.

## Данные

iNaturalist Open Dataset, бакет `s3://inaturalist-open-data/`. Регион - Вьетнам, `place_id=6847`.
